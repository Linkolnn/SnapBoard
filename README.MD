**SnapBoard**
SnapBoard - визуальная доска вдохновения для создания коллекций изображений. Пользователи могут загружать картинки, организовывать их в тематические доски (дизайн, путешествия, рецепты, мода) и делиться коллекциями. Masonry layout обеспечивает красивое отображение изображений разных размеров.

Основной функционал:
- Регистрация и аутентификация пользователей
- Создание тематических досок/коллекций
- Загрузка изображений (drag & drop или по URL)
- Masonry grid layout с бесконечным скроллом
- Поиск по коллекциям и тегам
- Модальные окна с детальным просмотром
- Добавление описаний и заметок к картинкам
- Светлая/темная тема

docker-compose -f docker-compose.dev.yml up -d

Технологии:
Frontend:
- Nuxt 4 - фреймворк для Vue.js
- Pinia - state management
- Sass - препроцессор CSS
- nuxt-security - модуль безопасности
Backend:
- Node.js - серверная платформа
- NestJs - веб-фреймворк
- PostgreSQL - реляционная база данных
- Sharp - обработка изображений
Безопасность:
- JWT - токены аутентификации (access + refresh)
- bcrypt - хеширование паролей
- helmet - защита HTTP заголовков
- express-rate-limit - защита от DDoS/брутфорса
- express-validator - валидация входных данных
- cors - настройка CORS политик
- hpp - защита от HTTP Parameter Pollution
- express-mongo-sanitize - защита от injection
- multer - безопасная загрузка файлов
DevOps:
- Docker - контейнеризация
- Docker Compose - оркестрация контейнеров
